Our first founder is Tony Ivanov from PicoStack. PicoStack is a modular P2P development kit, it allows you to build progressive web apps without any servers.

In this session, Tony will be covering the basics of PicoStack, including: handling local storage, replication and peer discovery.
The demo is a minimal to-do app. You can try it here: https://pico-todo.surge.sh/

Main takeaways

  • 00:25 PicoStack to-do app intro
  • 2:06 Data Replication between multiple peers
  • 2:17 Creating pico-blocks (data units)
  • 3:30 Consensus Model (simple model, all blocks are accepted)
